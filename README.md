# PPMI T2 대상자 선정 작업

이 폴더는 PPMI T2 영상 다운로드 대상자를 고유 피험자 기준으로 정리한 작업 공간입니다.

## 폴더

- `outputs/`: 분석 결과와 다운로드 대상 목록이 저장됩니다.
- `node_modules/`: XLSX 생성에 사용한 로컬 도구 연결 폴더입니다. 분석 결과가 아니므로 열거나 수정할 필요가 없습니다.

## 파일

- `build_t2_workbook.mjs`: CSV 결과를 읽어 요약 시트와 다운로드 목록이 포함된 XLSX 파일을 만드는 스크립트입니다.

## 주요 결과

최종 결과는 `outputs/t2_303_unique_subjects/`에 있습니다. 실제 확인과 다운로드 작업에는 해당 폴더의 XLSX 파일을 우선 사용합니다.
