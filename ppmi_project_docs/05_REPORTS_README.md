# 05_REPORTS_ARCHITECTURE_AND_QC

프로젝트 설명과 발표에 사용하는 최종 시각 자료입니다.

- `t2_preprocessing_architecture.png`: 실제 단계별 MRI가 포함된 최종 아키텍처 이미지
- `t2_preprocessing_architecture.pptx`: 발표용 단일 슬라이드
- `t2_preprocessing_architecture.svg`: 아이콘 기반 벡터 레이아웃 원본
- `render_t2_architecture.py`: 아키텍처 PNG 생성 코드

최종 PNG와 PPTX의 MRI 예시는 `sub-3008_I366281`의 실제 전처리 결과를 사용했습니다. SVG는 레이아웃 편집용 기반 파일이며 실제 MRI가 포함된 최종본은 PNG/PPTX입니다.
